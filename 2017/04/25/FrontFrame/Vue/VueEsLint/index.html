ESLint

发表于 2017-04-25 | 分类于 FrontFrame

　　ESLint是一个专为 Vue.js 应用程序开发的状态管理模式。它采用集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以一种可预测的方式发生变化。

ESLint 一个 JavaScript 代码的静态检测工具，相比之前比较流行的 JSHint，ESLint 扩展性强、配置灵活、支持 ES6 …
